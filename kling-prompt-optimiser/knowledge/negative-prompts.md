# Negative Prompt Templates

Pre-built negative prompts organised by scenario. Select the most relevant template and customise as needed.

## General Purpose

```
motion blur, face distortion, warping, morphing, inconsistent physics, floating objects, unnatural movements, extra limbs, background shifting, temporal flickering, colour banding
```

## Human Subjects

```
deformed hands, incorrect finger count, asymmetrical facial features, unnatural joint angles, extra fingers, disfigured hands, sliding feet, floating limbs
```

## Serious / Gritty Tone

Counteracts Kling's optimism bias — the model tends toward bright, happy outputs:

```
smiling, laughing, cartoonish, bright colours, low resolution, morphing, blurry text, disfigured hands, extra fingers
```

## Corporate / Professional

```
glasses, sunglasses, facial hair, beard, changing clothes, suit colour shift, missing tie, open collar, messy hair, sweat, skin changes, de-aging, fewer wrinkles, moving desk items, extra fingers, bad hands
```

## Fantasy / Period

```
modern clothing, sneakers, shifting armor plating, morphing sword hilt, changing cape colour, disappearing scars, shifting tattoos, flickering jewelry, modern background elements, distorted weapon, changing hair length
```

## Product / Brand

```
logo distortion, text warping, brand colour shifts
```

## Anti-Plastic (Combat Over-Smoothing)

```
smooth plastic skin, 3D render, over-smoothed textures
```

## Usage Rules

- Be specific — vague negatives like "bad quality" or "ugly" have no effect
- Combine templates when relevant (e.g., Human Subjects + Serious Tone for a dramatic character scene)
- Keep negative prompts shorter than the main prompt — they're guardrails, not the creative direction
