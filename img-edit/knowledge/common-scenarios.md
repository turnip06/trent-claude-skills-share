# Common Edit Scenarios

### Scenario 1: "Change the lighting to golden hour"

**Approach:** Comprehensive lighting overhaul
- Specify new colour temperature (3000-3500K)
- Modify shadow direction and length
- Add warm highlights, cool shadow complements
- Adjust overall colour grading
- Maintain composition and elements

### Scenario 2: "Remove the person in the background"

**Approach:** Clean removal with intelligent fill
- Identify what should replace removed element
- Match texture/pattern from adjacent areas
- Preserve lighting logic in filled area
- Ensure seamless edge blending
- Maintain perspective consistency

### Scenario 3: "Combine these three images"

**Approach:** Strategic layering with unification
- Identify which image provides environment base
- Determine lighting hierarchy
- Generate all necessary new shadows
- Unify depth of field across sources
- Apply consistent colour grade

### Scenario 4: "Add rain to this scene"

**Approach:** Atmospheric enhancement
- Add visible rain streaks with motion blur
- Wet all surfaces with reflections
- Modify lighting (slight diffusion, colour shift)
- Add atmospheric haze reducing distant contrast
- Generate rain-specific effects (puddles, mist)
