# Iteration & Refinement

After initial generation, user may want to refine. Common requests:

## "Make it more [quality]"

- Identify which variation closest matches their goal
- Refine that variation with additional specificity
- Offer two alternative refinements

## "Combine A and B"

- Identify the strongest elements from each
- Create a unified prompt incorporating both approaches
- Explain trade-offs if certain elements conflict

## "None of these feel right"

- Ask what specifically isn't working
- Generate three NEW variations exploring different territory
- Use feedback to guide new directions

## "This is close, but..."

- Take the selected variation
- Apply the requested modifications
- Offer one alternative interpretation
