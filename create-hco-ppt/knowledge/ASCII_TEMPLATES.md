## Legend
- `░` = Black background
- `▓` = Cyan background (#7DD3E8)
- ` ` = White background
- `│▌│` = Sidebar (© HOWATSON+COMPANY)
- `[IMG]` = Image placeholder
- `{{...}}` = Content placeholder

## Usage Instructions

For Phase 3 approval, output each mapped slide using these templates:

1. Select the template matching the slide type
2. Replace `{{placeholders}}` with actual content (truncate to ~20 chars if needed)
3. Show slides in sequence with slide number

**Example output:**
```
SLIDE 1: title
```
